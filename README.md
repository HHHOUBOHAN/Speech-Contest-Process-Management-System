演讲比赛流程管理系统（演讲管理系统）
本项目是一个基于C++开发的命令行应用程序，旨在自动化管理演讲比赛的整个流程，包括参赛者创建、多轮比赛抽签、评分、结果显示以及历届数据的持久化存储。

?? 项目简介
该模拟了一个包含两轮筛选系统的演讲比赛：

第一轮：所有参赛选手参加，每6人一组进行比赛，每组前3名晋级。

第二轮（决赛）：晋级选手进行最终对决，决出冠、亚、季军。

数据管理：比赛结果会自动保存到speech.csv文件中，支持随时查看往届记录。

?? 功能模块
演讲比赛开始：执行抽签、比赛评分、晋级逻辑。

查看往届记录：从 CSV 文件中加载并删除显示历届前三名信息。

清空重置记录：重置存储文件，清空所有历史数据。

退出系统：安全释放资源并退出程序。

?? 文件结构
纯文本
├── main.cpp            # 程序入口，控制主菜单循环
├── speaker.h           # 选手类头文件，定义选手属性
├── speaker.cpp         # 选手类实现（预留）
├── speechManager.h     # 核心管理类头文件，定义流程控制接口
├── speechManager.cpp   # 核心管理类逻辑实现（评分、保存、加载等）
└── speech.csv          # 数据存储文件（程序运行时自动生成）
?? 核心逻辑说明
1.比赛流程
系统通过speechManager::startSpeech()驱动整个比赛流程：

speechDraw()：使用std::random_shuffle实现随机抽签。

speechContest()：10位评委打分，实现最高/最低分取基层。

saveRecord()：将结果以分隔值（CSV）格式写入磁盘。

2.数据解析（loadRecord）
系统通过手写字符串解析逻辑处理CSV文件：

使用string::find()定位逗号。

使用string::substr()精确截取数据字段。

std::map<int, vector<string>>在内存中使用管理历届数据。
